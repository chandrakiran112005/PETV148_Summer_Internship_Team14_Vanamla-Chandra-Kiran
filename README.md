# Password Policy Compliance Auditor

## Project Overview

The Password Policy Compliance Auditor is a web-based cybersecurity application developed using Python, Flask, SQLite, HTML, CSS, JavaScript, and bcrypt.

The application analyzes password strength based on security rules inspired by NIST SP 800-63B and OWASP Password Guidelines. It provides password scoring, security recommendations, password history, and compliance reports while securely storing password information using bcrypt hashing.

## Features

- Password Strength Analysis
- NIST Compliance Evaluation
- OWASP Compliance Evaluation
- Password History
- Compliance Reports
- SQLite Database
- bcrypt Password Hashing
- Responsive Web Interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- bcrypt
